# 🌍 Global Development Categorization Dashboard

ML-powered Streamlit app for clustering countries by socio-economic indicators using KMeans.

---

## 📁 Project Structure

```
your-repo/
├── app.py                              # Main Streamlit application
├── train_model.py                      # Preprocessing pipeline trainer
├── World_development_mesurement.xlsx   # Raw dataset
├── requirements.txt                    # Python dependencies
├── .streamlit/
│   └── config.toml                     # Streamlit config
└── model/                              # Generated by train_model.py
    ├── imputer.pkl                     # SimpleImputer object
    ├── scaler.pkl                      # StandardScaler object
    └── features.pkl                    # Feature list
```

---

## 🚀 Deployment Steps

### **Step 1: Generate Model Files Locally**

Before pushing to GitHub, run the training script on your machine:

```bash
python train_model.py
```

**Expected output:**
```
Loading data...
✅ Loaded 2704 rows, 25 columns
...
✅ features.pkl saved

🎉 TRAINING COMPLETE!
Files saved in 'model/' directory
```

This creates the `model/` folder with 3 pickle files.

---

### **Step 2: Push to GitHub**

Upload these files to your repository:

```
✅ app.py
✅ train_model.py
✅ requirements.txt
✅ World_development_mesurement.xlsx
✅ .streamlit/config.toml
✅ model/imputer.pkl
✅ model/scaler.pkl
✅ model/features.pkl
```

**⚠️ CRITICAL:** Make sure the `model/` folder is uploaded. Without it, deployment will fail.

---

### **Step 3: Deploy on Streamlit Cloud**

1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Connect your GitHub account
3. Select your repository
4. Set:
   - **Main file path:** `app.py`
   - **Python version:** 3.10 or higher
5. Click **Deploy**

---

### **Step 4: Deploy on Render (Alternative)**

1. Go to [render.com](https://render.com)
2. Create a new **Web Service**
3. Connect your GitHub repo
4. Settings:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `streamlit run app.py --server.port=$PORT --server.address=0.0.0.0`
5. Click **Create Web Service**

## 📊 Features

- **Interactive Clustering:** Adjust k (2-6 clusters) via slider
- **World Map Visualization:** Choropleth map of development levels
- **Country Drilldown:** Detailed analysis per country
- **Radar Chart:** Compare country vs cluster average
- **Distribution Plots:** Box plots for indicator analysis
- **CSV Export:** Download country reports

## 🔧 Technologies

- **Python 3.10+**

## 📝 Dataset

Source: World Development Indicators (2704 countries/years)

**Features (22 indicators):**
- Economic: GDP, Tax Rates, Business Metrics
- Social: Life Expectancy, Birth Rate, Education
- Environmental: CO2 Emissions, Energy Usage
- Infrastructure: Internet, Mobile Phone Usage

## ⚠️ Common Issues

### **Error: Model files not found**
**Fix:** Run `train_model.py` locally before deploying.

### **Error: Module not found**
**Fix:** Ensure `requirements.txt` is in the root directory.

### **Error: Excel file not found**
**Fix:** Verify `World_development_mesurement.xlsx` is uploaded to GitHub.
